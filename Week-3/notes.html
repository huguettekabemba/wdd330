Week 3 Notes

Winter 2023

This week I studied objects, properties and methods and found a great example on W3 Schools. It talked about how cars have properties like weight and color, and methods like start and stop. These can be looked at like code. I'll include the code examples below.

Object Properties Methods
car car.name: "Fiat" car.start()
car.model: "500" car.drive()
car.weight: "850kg" car.brake()
car.colo: "White" car.stop()

Objects are variables, but they can contain many values. Here is some code for a computer.

let computer = {type:"Apple", model:"Macbook Pro", color:"Space Grey"};

They can be accessed by simply typing computer.type. This would target the type of the computer variable which would be Apple.

this always refers to the object it belongs to. It in itself is not given a value. It's value depends on where it is placed. Here is a code snippet from W3 Schools:

var person = {
firstName: "John",
lastName : "Doe",
id : 5566,
fullName : function() {
return this.firstName + " " + this.lastName;
}
};

Because this is directly after the person object, that object is the owner of the function that uses this.
