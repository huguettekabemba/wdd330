The functions above look fairly similar but have their differences. I understood the first one easily because it was the first one I learned. The second one helps me to understand the third. The ones that make the most sense to me are the first and the third. I do not understand why you would name a function if you wont be referencing that name later on.

Something I found very interesting was the use of methods. Methods are functions that are built in to Javascript. Therefore, you do not need to code them, instead, you just call them. I have used many before life console.log and Math.pow but was unaware of the many that there are. Below I will include some that can be used to manipulate and change arrays. the .push method can add something to the end of and array while the .pop method returns and and removes the last value in the array referenced.

let sequence = [1, 2, 3]; sequence.push(4); sequence.push(5); console.log(sequence); // → [1, 2, 3, 4, 5] console.log(sequence.pop()); // → 5 console.log(sequence); // → [1, 2, 3, 4]
