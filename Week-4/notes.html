Week 4 Notes

Winter 2023

In Object-oriented Programmming there are three main concepts.

  1. Encapsulation - Parts of the program are inside objects and only essential functionalities are exposed to the end user.
  2. Polymorphism - The same process can be used for different objects. Various objects can be used for the same method.
  3. Inheritance - You can take an object that already exists and Inherit all its properties and methods.

A real world example I can think of is a lamp. There's a lot to a lamp but all we see is a simple switch that turns it on or off.(Encapsulation). Most, if not all, lamps can be plugged into all outlets and use any type of light bulb. (Polymorphism) Lamps are all built similarly but some can have differences. (Inheriitance)

I thnk that the benefit of Object oriented programming is having flexible code. You can create an object and use it in other proograms.

const Dice = function(sides=6){
this.sides = sides;
this.roll = function() {
return Math.floor(this.sides * Math.random() + 1)
}
};
